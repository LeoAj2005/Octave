#  Octave

A modern music player for WebOS focused on simplicity, performance, and a smooth listening experience.

## Status

🚧 **Pre-Alpha** — Active Development

### Planned Features

* Audio Playback
* Music Library Management
* Playlists
* Album Artwork
* Metadata Support

## Installation

```bash
git clone https://github.com/LeoAj2005/Octave.git
cd Octave
```
